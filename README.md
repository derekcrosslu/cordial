# CORDIAL

Project Cordial is an innovative web project designed to showcase the unique ambiance of Cordial, a wind bar located in Barranco, Peru. This digital representation brings to life the restaurant's aesthetic and atmosphere through a stunning 3D online experience. Developed using React, React Fiber, and Three.js for the frontend, the website captures the essence of Cordial with interactive and immersive 3D visuals. On the backend, the project employs RedwoodJS, ensuring a seamless and efficient user experience. This project highlights the intersection of technology and hospitality, offering visitors a virtual taste of what Cordial has to offer before they even step through its doors.
